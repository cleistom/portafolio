Acerca del Proyecto

Este editor de código en línea es una aplicación web desarrollada con JavaScript, HTML5 y Tailwind CSS. Utiliza la librería CodeMirror para proporcionar un editor de código con resaltado de sintaxis en tiempo real. Soporta múltiples lenguajes, como JavaScript, HTML y CSS.

Las tecnologías utilizadas en este proyecto son:

JavaScript

Lógica del editor

Tailwind CSS

Diseño e interfaz

CodeMirror
